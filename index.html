TEA HISTORY

PUSAT KRONOLOGI RESMI PERJALANAN TEA

SEKTOR 01 — AWAL MULA TEA

Awal mula TEA ketemu para admin awalnya di TikTok. Pada tanggal 18 Desember 2025, TEA resmi terbentuk. Admin favorit TEA bertemu di TikTok. Admin Dyel menjadi member ke-6 dan Admin Science menjadi member ke-9. Admin Science masuk karena ingin menjadi admin di marga, karena belum pernah jadi admin di marga Astronomy. Admin Dyel masuk karena diajak oleh Owner. Admin Science menulis di tag anggota: "Cita cita ku ingin menjadi admin TEA." Owner merespon: "Kalau mau jadi admin sering sering on aja." Owner berkata, "Sip saya catat." Admin Science menjawab, "Sip min." Keesokan harinya Owner berkata kalau mau jadi admin harus promosi marga TEA. Admin Science pun promosi di akun TikTok-nya, dan Admin Dyel promosi di akun YouTube-nya. Itulah awal mula admin favorit TEA bertemu.

SEKTOR 02 — KRISIS EDITOR

Admin Doni berkata, "Ada yang bisa buat intro?" Tidak ada yang menjawab. Admin Slamet berkata, "Buat wm aja dulu yang gampang." Karena TEA krisis admin editor, Admin Dyel membuat watermark khusus TEA. Admin Dyel menjadi admin editor TEA dan mencoba membuat intro khusus TEA. Sampai sekarang Dyel masih membuat intro untuk marga TEA. Di situlah Admin Dyel menjadi editor sekaligus penyelamat TEA.

SEKTOR 03 — ERA 20 MEMBER

Masih ada admin old seperti Admin Slamet, Admin Doni, Admin Dyel, Admin Science, Admin Alya, dan Admin Kaizou. Walaupun hanya 20 member, chat sangat seru walau kadang sepi. Owner promosi di TikTok, tidak disangka langsung ramai. Dari 20 naik 30, setelah tahun baru naik 100 dalam beberapa minggu. Admin Science dan Admin Dyel terus promosi sampai 200 member. Di 200 member mulai seret.

SEKTOR 04 — KONFLIK ADMIN

Saat member masih 20 ada Admin Balqis yang jarang on tapi tidak ingin turun jabatan. Kalau ada soal Astronomy selalu diremehkan. Saat memberi soal ke member baru berkata, "Di rodok mau?" Admin lain tidak suka. Setelah vote turun jabatan, Balqis marah dan berkata kasar. Admin Science, Admin Dyel, dan Admin Slamet berdebat. Karena kesal, Admin Dyel langsung kick tanpa basa basi.

SEKTOR 05 — ERA ADMIN SHADOW

Sebelum 100 member, chat sangat ramai karena Admin Shadow sering bahas Astronomy. Ia ingin menjadi admin dan membuat grup pembelajaran. Admin Science berkata, "Sering sering on aja." Shadow tetap mengajar walau belum admin. Saat pemilihan, Shadow menjadi admin dan membuat grup pembelajaran. Di era itu TEA sangat ramai dan banyak yang suka Shadow.

SEKTOR 06 — KELUARNYA ADMIN SCIENCE

Saat 200+ member chat sepi. Banyak sider. Admin Science keluar karena ingin marga yang ramai. Jabatan jatuh ke Admin Dyel menjadi Co-Owner baru. Member dan admin sedih kehilangan Admin Science.

SEKTOR 07 — REVOLUSI TEA

Tanggal 18 Februari setelah buka puasa, Admin Science kembali untuk revolusi TEA. Admin Putra memberi saran membuat sistem dan peraturan baru. Admin Science mengubah sistem. Beberapa hari kemudian TEA sangat ramai kembali karena Admin Science dan Admin Dyel.

SEKTOR 08 — MEGA COLLAB 300 MEMBER

Saat 300 member ada Mega Collab melibatkan 30 member. Sangat capek mengumpulkan PP. Admin Science, Admin Dyel, dan admin lain berjuang sampai terkumpul. Admin Dyel sangat kelelahan mengedit tapi rela demi TEA.

SEKTOR 09 — LONJAKAN KE 500+

Admin Science berkata ke Admin Dyel, "Coba promosiin biar TEA rame." Admin Dyel mengedit capek capek demi TEA. Saat diupload benar benar ramai. Dari 300+ naik 500+. Chat setiap hari selalu jebol.
